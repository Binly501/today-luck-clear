# 15 分钟快速启动指南

> 不废话，直接跑起来

---

## 步骤 1：下载 HBuilderX（3 分钟）

https://www.dcloud.io/hbuilderx.html

- 下载"标准版"
- 解压后双击运行

---

## 步骤 2：打开项目（1 分钟）

1. HBuilderX 菜单：**文件 → 打开目录**
2. 选择 `today-luck-clear` 文件夹

---

## 步骤 3：配置 AppID（2 分钟）

1. 打开 `manifest.json`
2. 找到 `"appid": "填入你的小程序 AppID"`
3. 替换为你的真实 AppID

**没有 AppID？**
- 临时用 `wx1234567890abcdef` 也能运行
- 真机预览需要真实 AppID

---

## 步骤 4：临时处理图标（2 分钟）

打开 `pages.json`，找到 `tabBar` 部分，**整体注释掉**：

```json
{
  "pages": [...],
  "globalStyle": {...}
  // "tabBar": { ... }  ← 把这整段注释掉
}
```

> 这样就能先跑起来，图标后面再补

---

## 步骤 5：运行到微信开发者工具（5 分钟）

1. HBuilderX 菜单：**运行 → 运行到小程序模拟器 → 微信开发者工具**
2. 第一次会提示选择微信开发者工具路径
3. 等待编译完成（10-30 秒）
4. 看到模拟器显示首页 = 成功！

---

## 步骤 6：真机预览（2 分钟）

1. 微信开发者工具右上角点击 **预览**
2. 用手机微信扫码
3. 在手机上测试完整流程

---

## 常见问题

### ❌ 提示"manifest.json 格式错误"

检查 JSON 格式，确保：
- 没有多余的逗号
- 所有引号都是双引号

### ❌ 真机预览打不开

1. 检查 AppID 是否正确
2. 微信开发者工具 → 详情 → 本地设置
3. 勾选"不校验合法域名..."

### ❌ 黑屏/白屏

1. 打开微信开发者工具的 **调试器**
2. 查看 Console 中的错误信息
3. 截图发给我帮你解决
