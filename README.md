# 今日晦气清除 - 微信小程序

> 极简情绪解压工具，写下不开心，然后清除它

## 项目信息

- **产品名称**: 今日晦气清除
- **类型**: 微信小程序
- **开发框架**: uni-app
- **周期**: 1～2 周
- **核心**: 极简、轻量、无后端、纯前端

## 功能特点

✅ 4 个核心页面：首页、清除动画页、历史页、我的页  
✅ 7 种清除方式：烧掉、吹走、丢垃圾桶、净化、冲走、撕碎、踩碎  
✅ 每日限制 3 次，0 点自动重置  
✅ 看广告/分享可获取额外次数  
✅ 本地存储，不上传用户隐私内容  
✅ 治愈系 UI 设计，简约干净

## 快速开始

### 1. 环境准备

- 安装 HBuilderX（推荐）或 Vue CLI + uni-app 插件
- 注册微信小程序账号，获取 AppID

### 2. 导入项目

```bash
# 使用 HBuilderX
1. 打开 HBuilderX
2. 文件 -> 导入 -> 从本地目录导入
3. 选择 today-luck-clear 目录
```

### 3. 配置小程序

编辑 `manifest.json`，填入你的小程序 AppID：

```json
{
  "mp-weixin": {
    "appid": "你的小程序 AppID"
  }
}
```

### 4. 运行/发布

```bash
# 在 HBuilderX 中
1. 运行 -> 运行到小程序模拟器 -> 微信开发者工具
2. 发行 -> 微信小程序
```

### 变现逻辑

1. **免费用户**: 每天 3 次基础次数，有广告
2. **激励视频**: 观看广告 +1 次/天
3. **分享好友**: 分享 +1 次/天（最多 2 次）
4. **付费去广告**: ¥6 永久去除广告
